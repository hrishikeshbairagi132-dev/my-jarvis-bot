# JARVIS Trading Dashboard 👑

यह एक एडवांस्ड ट्रेडिंग डैशबोर्ड है जो Yahoo Finance डेटा, GNews सेंटीमेंट एनालिसिस, और रीयल-टाइम टेलीग्राम अलर्ट्स का उपयोग करके पेपर ट्रेडिंग (Paper Trading) में मदद करता है।

## फीचर्स:
* 📈 **Technical Indicators:** RSI, ADX, EMA20, VWAP, ATR.
* 📰 **News Sentiment:** GNews API का उपयोग करके बाजार का मूड समझना।
* 🔔 **Telegram Alerts:** ट्रेड सिग्नल सीधे आपके टेलीग्राम पर।
* 📉 **Charts:** Plotly का उपयोग करके रीयल-टाइम इंटरैक्टिव चार्ट्स।

## कैसे सेटअप करें (Deployment):
1. **GitHub Repository:** इस कोड को GitHub पर रखें।
2. **Streamlit Cloud:** [share.streamlit.io](https://share.streamlit.io/) पर जाएं।
3. **Secrets:** 'Advanced Settings' में ये कीज़ डालें:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `GNEWS_API_KEY`
